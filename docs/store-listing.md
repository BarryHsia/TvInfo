## App Name
Tv Info - System Dashboard

## Short Description (80 chars max)
View your Android TV's hardware, software, network & storage info at a glance.

## Full Description
Tv Info is a sleek system information dashboard designed for Android TV and Google TV devices. Get a comprehensive overview of your TV's hardware and software details through a beautiful, dark-themed interface optimized for remote control navigation.

📺 WHAT YOU CAN SEE:

• Overview — Device profile, CPU cores, memory usage, and network status
• Hardware — Processor architecture, device model, RAM details, supported ABIs
• Display — Screen resolution, refresh rate, color depth, pixel density
• Network — Connection type, IP address, gateway, DNS, MAC address, link speed
• Software — Android version, API level, security patch, SELinux status, kernel info, build fingerprint
• Storage — Internal storage usage breakdown, external storage detection, file system type

🎮 DESIGNED FOR TV:
• Full D-pad remote control support
• Netflix-style collapsible sidebar navigation
• Smooth animations and focus indicators
• No touch screen required

🎨 BEAUTIFUL UI:
• Dark sci-fi themed dashboard
• Bento grid card layout
• Real-time clock display
• Animated progress bars and status indicators

🔒 PRIVACY FIRST:
• All data is read locally on your device
• No personal data is collected or transmitted
• Anonymous usage analytics only (can be disabled)

Perfect for tech enthusiasts, developers, and anyone who wants to know what's inside their Android TV device.
